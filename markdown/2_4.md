**School.py**

``` {.python}
# This is not quite true outside of USA
# and is based on my dim memories of my younger years
print("Firstish Grade")
print("1 + 1 =", 1 + 1)
print("2 + 4 =", 2 + 4)
print("5 - 2 =", 5 - 2)
print()
print("Thirdish Grade")
print("243 - 23 =", 243 - 23)
print("12 * 4 =", 12 * 4)
print("12 / 3 =", 12 / 3)
print("13 / 3 =", 13 // 3, "R", 13 % 3)
print()
print("Junior High")
print("123.56 - 62.12 =", 123.56 - 62.12)
print("(4 + 3) * 2 =", (4 + 3) * 2)
print("4 + 3 * 2 =", 4 + 3 * 2)
print("3 ** 2 =", 3 ** 2)
```

Output:

`Firstish Grade`\
`1 + 1 = 2`\
`2 + 4 = 6`\
`5 - 2 = 3`\
\
`Thirdish Grade`\
`243 - 23 = 220`\
`12 * 4 = 48`\
`12 / 3 = 4`\
`13 / 3 = 4 R 1`\
\
`Junior High`\
`123.56 - 62.12 = 61.44`\
`(4 + 3) * 2 = 14`\
`4 + 3 * 2 = 10`\
`3 ** 2 = 9`
